brain hacks
curiosity + eustress
all stages of problem solving

problem solving usually moves through four phases:
> preparation — understand / define the problem
> incubation — generate / let ideas simmer
> illumination — insight / aha moment
> verification — test / refine / ship

curiosity = drive to explore
eustress = useful stress that energizes instead of drains

1. preparation

reframe with better questions

> what if this limit didn’t exist?
> how would someone from a different field see this?
> what’s the smallest interesting version of this problem?
(small dopamine hits from tiny progress)

slow nasal breathing reset

> 4 sec in — 6–8 sec out
> repeat 8–12 cycles before starting
> lowers threat mode → opens curiosity mode

2. incubation

play + gamify the mess

> set 7-minute timer → write 12 stupid ideas
> give points for the weirdest ones
> reward = coffee / stretch / music

diffuse mode walks

> leave the desk
> walk / shower / dishes / stare at wall
> let random connections form
> best ideas usually arrive when you’re not forcing them

3. illumination

obstacle removal visualization

> imagine the biggest blocker is gone
> what changes immediately?
> run the thought experiment 2–3 times

perspective switch

> how would a 12-year-old solve it?
> how would a sci-fi character do it?
> how would your future self (5 years wiser) approach it?
> quick vagus hack: hum or sing low for 30 seconds

4. verification

hierarchical + what-if breakdown

> solve one small piece first
> ask: what if I’m wrong here?
> what’s the cheapest/fastest way to test this assumption?

gratitude anchor

> after testing write three things:
- one thing you learned
- one thing that worked
- one thing you’re glad you tried
> closes loop → sustains motivation

start with 1–2 tricks per stage
track which ones actually make you feel
curious + alert instead of anxious + stuck
repeat → brain rewires
